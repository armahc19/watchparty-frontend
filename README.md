# Welcome to your Watch Party

## Project info

Watch movies & shows together in real-time – completely free.
Flying reactions, chat, perfect sync – feels like Netflix + Discord had a baby.

Features

- Real-time video sync (play, pause, seek)
- Flying reactions (Popcorn, Fire, Heart, Laugh, Sleep, Broken Heart)
- Flying chat messages (big, beautiful, cinematic)
- Up to 5 people per room (free)
- 30-minute rooms (auto-cleaned)
- Upload up to 3 files
- Mobile-ready (looks insane on phone)
- VIP rooms available (no limits – DM me)
